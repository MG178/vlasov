> **FastDelivery** — учебный MVP для автоматизации курьерской доставки малогабаритных грузов. Проект собран как локальное WPF-приложение с ролевым доступом, заказами, курьерами, тарифами, отчетами и уведомлениями.

## Что умеет приложение

- вход по демо-аккаунтам;
- создание и редактирование заказов;
- назначение курьера на заказ;
- смена статусов доставки без QR-кодов;
- управление списком курьеров;
- редактирование зон и тарифов;
- просмотр истории заказа;
- уведомления по ролям;
- сводка и экспорт отчета в CSV.

## Демо-аккаунты

- `admin` / `admin`
- `dispatcher` / `dispatcher`
- `courier1` / `courier1`
- `client1` / `client1`

## Интерфейс

В приложении доступны вкладки:

- Заказы;
- Курьеры;
- Справочники;
- Отчеты;
- Уведомления.

## Ролевой доступ

- Клиент видит заказы, относящиеся к нему.
- Курьер видит назначенные заказы.
- Диспетчер и администратор видят все рабочие экраны.

## Технологии

- WPF на .NET 8;
- локальное хранилище данных в памяти для MVP;
- SQL Server-ориентированная модель данных сохранена в коде как основа для последующей интеграции.

## Запуск

1. Откройте решение в Visual Studio.
2. Проверьте строку подключения в `appsettings.json`.
3. Запустите проект `FastDelivery`.

## Примечание

QR-коды и серверные триггеры из полного ТЗ в этом MVP не реализованы как обязательный пользовательский сценарий. Приложение сфокусировано на базовом рабочем процессе доставки и демонстрации интерфейса.
